# Minimal Static Web App

This repository contains a set of files designed to be published as a public GitHub Pages site.

## Files Included

- `index.html` - The homepage providing links and explanations of each file.
- `ashravan.txt` - A short story by Brandon Sanderson about Ashravan's fate after Shai restores him.
- `dilemma.json` - JSON data describing moral choices for an autonomous vehicle dilemma.
- `about.md` - A brief three-word personal description.
- `pelican.svg` - An SVG illustration of a pelican riding a bicycle.
- `restaurant.json` - A recommended restaurant in Mumbai.
- `prediction.json` - Prediction for the Federal Reserve Funds rate by December 2025.
- `LICENSE` - The MIT License text.
- `uid.txt` - UID file as provided.

## Usage

Open `index.html` in a web browser to navigate the contents easily. All files are static and do not require a server.
